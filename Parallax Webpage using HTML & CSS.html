3 Days FDP on Full Stack Web Development

Objectives: Use the learned skills, knowledge and abilities to develop web sites Apply basic design principles to present ideas, information, products, and services on websites Apply basic programming principles to the construction of websites Effectively manage website projects using available resources and know about digital marketing

Learning outcomes

Knowledge of Internet, Domains, Networking Understanding architecture of a website Core JAVA (J2SE) Programming and Coding Skills Work with most popular web development technologies Create and host websites and web apps for commercial usage Create responsive and interactive UI Work with React JS framework to create responsive web applications Fundamentals of Databases Work with databases in web apps

Eligibility for Faculty Certification

All the faculty members participating in the program will have to meet the following criteria: 70% attendance is mandatory for obtaining the e-certificate for FDP  Submission of pre assessment, and post assessment is mandatory for obtaining the e-certificate for FDP
